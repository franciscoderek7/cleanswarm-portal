N8N WORKFLOWS

Core Automation Pipelines

Self-hosted n8n instance handles all business logic. Zero recurring API costs. Full data ownership.

TRIGGER: New Job Created

Job Assignment & Dispatch

Auto-assigns cleaners based on proximity, availability, and skill match. Sends SMS confirmation to client and cleaner.

→ Webhook receives new job from portal
→ Query Truein for available staff
→ Calculate proximity (Google Maps API)
→ Assign best-fit cleaner
→ Send SMS via Twilio (client + cleaner)
→ Update job status in database
TRIGGER: Truein Clock-Out

Automated Invoice Generation

When cleaner clocks out via Truein biometric, invoice auto-generates with hours worked, travel time, and supplies used.

→ Truein webhook: clock-out event
→ Calculate billable hours (minus breaks)
→ Pull job rate from client contract
→ Generate PDF invoice (template)
→ Email invoice to client
→ Update accounting ledger
TRIGGER: Daily 6:00 AM

Morning Dispatch Report

Daily briefing sent to operations manager with today's jobs, assigned staff, potential conflicts, and weather alerts.

→ Cron trigger: 6:00 AM EST daily
→ Query today's scheduled jobs
→ Check Truein: who's available
→ Flag conflicts/understaffing
→ Pull weather forecast (OpenWeather)
→ Send Slack/Email digest
TRIGGER: Invoice 7 Days Overdue

Payment Follow-Up Sequence

Automated payment reminders escalating from friendly reminder to formal notice. Reduces AR by 40%.

→ Check: invoice unpaid > 7 days
→ Day 7: Friendly email reminder
→ Day 14: SMS + email with late fee warning
→ Day 21: Phone call task to manager
→ Day 30: Formal notice + service pause
→ Update client risk score
TRIGGER: Job Completed

Quality Assurance & Review

Post-job quality check flow. Client receives satisfaction survey. Photos uploaded by cleaner are archived.

→ Cleaner uploads completion photos
→ Photos stored in S3 bucket
→ Client receives rating request (1-5)
→ If rating < 3: alert manager
→ Update cleaner performance score
→ Archive job record
TRIGGER: New LOI Signed

Client Onboarding Pipeline

Full onboarding automation: contract generation, Truein setup, first job scheduling, and welcome sequence.

→ LOI form submitted on portal
→ Generate contract PDF (DocuSign)
→ Create client in Truein system
→ Setup recurring job schedule
→ Send welcome email sequence (5 emails)
→ Assign account manager

BIOMETRIC ATTENDANCE

Truein Integration

Face-recognition attendance system for field cleaners. Eliminates time theft, automates payroll, and provides real-time location verification.

Why Truein for CleanSwarm

Face Recognition Clock-In

Cleaners clock in/out via phone camera at job site. GPS-verified location. No buddy punching possible.

Geofencing

Set 100m radius around each job site. Clock-in only works within geofence. Prevents false attendance.

Real-Time Dashboard

See who's on-site, who's late, who's done. Live map view of all active cleaners across all clients.

Automated Payroll

Hours auto-calculated. Overtime flagged. Export to QuickBooks/Wave. Zero manual timesheets.

Webhook API

Every clock-in/out fires webhook to n8n. Triggers job status updates, invoice generation, and alerts.

Pricing: $2/user/month

Free for up to 25 users. Scales to enterprise. No hardware needed — uses existing smartphones.

Truein Webhook Payload (Clock-Out Event)

// Fired when cleaner completes face-scan clock-out
POST /webhooks/truein/clock-out

{
  "event": "attendance.clock_out",
  "employee_id": "CS-007",
  "employee_name": "Sarah Johnson",
  "site_id": "site_ali_office_01",
  "clock_in": "2026-05-24T08:02:14Z",
  "clock_out": "2026-05-24T12:15:42Z",
  "hours_worked": 4.22,
  "location": {
    "lat": 44.3091,
    "lng": -78.3197,
    "within_geofence": true
  },
  "verification": "face_match_98.7%"
}

// n8n processes this to:
// 1. Calculate billable amount ($35/hr × 4.22 = $147.70)
// 2. Generate invoice PDF
// 3. Email to client (Ali's Cleaning Co.)
// 4. Update job status → "Completed"
